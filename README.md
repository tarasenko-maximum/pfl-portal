# AIMAX × PFL · Portal Pitch (Фаза 2)

Готовый к деплою одностраничный сайт-визитка для Prime Fighting League.

## Запуск локально
```bash
npm install
npm run dev
```

## Сборка
```bash
npm run build
```

## Деплой на Vercel
Просто выполните:
```bash
vercel --prod
```
Или подключите репозиторий через Vercel UI.

## Структура
- `src/components/` — 19 React-компонентов секций
- `public/images/` — медиа-ассеты и плейсхолдеры
- Стек: React 18, Vite, TailwindCSS, Framer Motion
